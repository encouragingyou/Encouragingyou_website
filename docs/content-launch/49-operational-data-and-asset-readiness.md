## Ready With An Explicit Public Boundary

| Surface | Public handling | Why the boundary exists |
| --- | --- | --- |
| Exact venue details | Shared on enquiry only | The current source set supports Rochdale publicly but not a blanket address. |
| Community-support delivery detail | Contact-led | The wider community route is real, but it does not have a standing public timetable. |
| Volunteer checks | Explained proportionately during first conversation | Different roles may need different checks; the site does not flatten that into one public promise. |
| Legal notices | Publish current launch truth only | Privacy, cookie, accessibility, and policy pages describe the live build without guessing future operations. |
| Trained-and-vetted safeguarding statement | Narrow public wording only | The brief supports the statement, but the site does not extend it into unverified procedural detail. |
