# Prompt 26 - Youth Club Trust And FAQ Composition

## Trust composition

The Youth Club route now spreads trust cues across the page instead of collapsing them into one generic safe-space claim.

Primary trust surfaces:

- hero supporting note that normalizes asking first
- at-a-glance copy that removes formal attendance pressure
- expectation cards that describe atmosphere and participation modes
- support cards that keep venue handling and safeguarding visible
- related-programme bridge that clarifies the programme/session boundary
- CTA band note that keeps safeguarding reachable at the end of the journey

## FAQ strategy

Prompt 26 introduces a dedicated `session-youth-club` FAQ group instead of reusing the generic joining-session answers.

The FAQ set is intentionally narrow:

- joining without already knowing anyone
- what the session may include
- what happens if someone feels quiet or unsure
- how a parent or carer can ask first

This keeps the FAQ useful without drifting into unsupported youth-work specifics.

## Editorial boundary

The route language is designed to feel warm and direct, but it still respects the public-proof boundary:

- no invented operational detail
- no vague wellbeing promises with no practical explanation
- no parent/carer framing that overwhelms the young-person pathway
- no generic community copy pasted from the wider programme page
